# expense-api


### Installation
1. Clone the repository
```bash
git clone https://github.com/Moses-93/expense-api.git
cd expense-api
```

2. Create and activate a virtual environment
```bash
python -m venv .venv
source .venv/bin/activate
```

3. Install dependencies
```bash
pip install -r requirements.txt
```
1
5. Start the server
```bash
bash scripts/run.sh
```
